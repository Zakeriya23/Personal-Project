My Projects

Discom - Discord Replit
GitHub Repository
Discom is a web application developed using the MERN stack (MongoDB, Express, React, Node.js) that emulates core functionalities of Discord. The frontend is built with React and Axios, providing a responsive user interface for real-time communication. The backend, powered by Express and MongoDB, manages user authentication and message handling. This project demonstrates proficiency in full-stack development and real-time data processing.

Finance Tracking App
GitHub Repository

This project strengthened my skills in Python programming, data management, and visualization.
